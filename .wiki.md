# Project Summary
The LRE HOLDINGS Inventory & Logistics Platform is a web application that enhances inventory management, logistics tracking, and user management for businesses. By integrating with Supabase for backend services, it offers a modern frontend experience that allows users to efficiently manage consignments, deliveries, and payments.

# Project Module Description
- **Backend**: Utilizes Supabase to manage data, authentication, and business logic.
- **Frontend**: Developed with React, this module provides an intuitive user interface for managing inventory, users, consignments, deliveries, and payments.
- **Dashboard**: Features role-based dashboards tailored for Admin, Affiliate, and Driver users to access their specific functionalities.

# File Description Inventory
- **backend/**: Contains backend logic with routers and services for handling requests.
- **frontend/**: Contains frontend code, including components and pages for user interaction.
- **main.py**: The entry point for the backend application.
- **App.tsx**: The main React component that sets up routing and application logic.
- **supabase.ts**: Library file for Supabase integration within the frontend.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend**: Supabase, Python (FastAPI)
- **Database**: Supabase PostgreSQL

# Usage
1. **Install Dependencies**:
   - Navigate to the frontend directory: `cd app/frontend`
   - Install dependencies: `pnpm install`
   
2. **Build the Application**:
   - Run the build command: `pnpm run build`

3. **Run the Application**:
   - Start the application server (local or remote setup instructions are omitted for brevity).
