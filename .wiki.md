# Project Summary
The LRE HOLDINGS Inventory & Logistics Platform is a web application designed to streamline inventory management, logistics tracking, and user management for businesses. It integrates with Supabase for backend services and utilizes a modern frontend stack to provide an intuitive user experience, enabling users to manage consignments, deliveries, and payments efficiently.

# Project Module Description
- **Backend**: Handles data management, authentication, and business logic using Supabase.
- **Frontend**: Built with React, this module provides a user interface for managing inventory, users, consignments, deliveries, and payments.
- **Dashboard**: Role-based dashboards for Admin, Affiliate, and Driver users to access relevant functionalities.

# File Description Inventory
- **backend/**: Contains the backend logic, including routers and services for handling requests.
- **frontend/**: Contains the frontend code, including components and pages for user interaction.
- **main.py**: The entry point for the backend application.
- **App.tsx**: The main React component that sets up routing and application logic.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend**: Supabase, Python (FastAPI)
- **Database**: Supabase PostgreSQL

# Usage
1. **Install Dependencies**:
   - Navigate to the frontend directory: `cd app/frontend`
   - Install dependencies: `pnpm install`
   
2. **Build the Application**:
   - Run the build command: `pnpm run build`

3. **Run the Application**:
   - Start the application server (local or remote setup instructions are omitted for brevity).
